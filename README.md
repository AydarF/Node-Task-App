Task Manager app. Built with:
- HTML, CSS, 
- Node.js, Express.js, 
- MongoDB, Mongoose,
- bcrypt (to encrypt passwords),
- jsonwebtokens,
- mutler (to upload files),
- sharp (to process images),
- SendGrid API (to send emails)


## Available Scripts

In the project directory, you can run:

### `npm install`

### `npm run dev`

Open http://localhost:3000/
